# 韩国 vs 捷克 · 数据之战 海报提示词

## 核心设计需求
- 体现两队球衣颜色（韩国深猩红 vs 捷克亮红+银）
- 融入六维度比分预测元素（艺术化呈现，避免直接博彩暗示）
- 欧冠之夜暗调奢华风格

-- 悬浮全息光粒子排列成抽象图案：
   1颗金色光点 vs 0颗暗点（1-0预测）

## 法律风险规避确认

| 禁用元素 | 替代方案 |
|---------|---------|
| FIFA/UEFA标志 | 抽象几何星形 |
| 国旗/国徽 | 抽象色块+纹理 |
| 球员肖像/面部 | 剪影/背影 |
| "赌博""投注""赔率"文字 | 纯数据可视化（粒子/光效） |
| 博彩公司LOGO | 无 |
| 具体奖金/金额 | 抽象"Kelly"数据流 |
| 比分数字直接写 | 粒子数量暗示（艺术化） |
